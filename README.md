## Overview

This project models voter engagement as a discrete-time Markov chain to study long-term behavioral patterns in civic participation.

## Motivation

Voter engagement is dynamic. Individuals transition between registration and participation states over time. Understanding these transitions helps identify long-run patterns of civic engagement.

## State Space

- U: Unregistered
- R: Registered but inactive
- A: Active voter
- I: Inactive voter
- D: Disenfranchised

## Methods

- Markov chain modeling
- Transition matrices
- Steady-state analysis
- Simulation
- Sensitivity analysis
